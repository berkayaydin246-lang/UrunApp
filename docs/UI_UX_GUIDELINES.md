# UI / UX Guidelines

## Target User
The app should be usable by 50+ users.

## Style
- Large text
- Clear buttons
- Minimal clutter
- Turkish-first labels
- Short explanations
- Clear color/status indicators

## Main Actions
Use simple labels:
- Ürün Ara
- Barkod Tara
- İçindekileri Tara
- Detayları Gör

## Result Screen
Show the result in this order:
1. Score label
2. One short summary
3. Important warnings
4. Ingredient details
5. Source / verification status

## Tone
Use clear and calm Turkish.

Good:
- "Bu ürün katkı maddesi içeriyor. Sık tüketilmesi önerilmez."

Avoid:
- "Bu ürün sağlığınız için çok zararlıdır."
- "Kesinlikle tüketmeyin."
- "Kanser yapar."

## Error Messages
Use helpful messages:
- "Ürün bulunamadı. Barkod okutmayı deneyebilirsin."
- "Barkod okunamadı. Kamerayı biraz yaklaştır."
- "İçindekiler yazısı okunamadı. Daha net bir fotoğraf çek."
